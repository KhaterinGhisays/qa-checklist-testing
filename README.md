# ✅ Checklist Testing Project

## 📌 Descripción
Proyecto de testing funcional utilizando checklist para validar el comportamiento de una aplicación web.

---

## 🎯 Objetivo
Ejecutar pruebas funcionales y verificar que las funcionalidades cumplan con los requisitos esperados.

---

## ✅ Actividades realizadas
- Creación de checklist de pruebas
- Ejecución manual de pruebas
- Registro de resultados (Passed / Skipped)
- Comparación entre resultado esperado y actual

---

## 🛠️ Herramientas
- Manual Testing
- Google Sheets

---

## 🔎 Tipos de pruebas
- Functional Testing
- UI Validation
